# Set new X parameter values

This method dispatches on the type of `xds_obj$XH_obj[[s]]`.

## Usage

``` r
change_XH_inits(xds_obj, i = 1, options = list())
```

## Arguments

- xds_obj:

  an **`xds`** model object

- i:

  the vector species index

- options:

  a named list

## Value

an `xds` object
