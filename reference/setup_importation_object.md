# Setup the Importation Object

Setup an object to handle malaria importation through travel and
visitors. Initializes the junction and zero-initializes the importation
parameters on the `XY_interface`.

## Usage

``` r
setup_importation_object(xds_obj)
```

## Arguments

- xds_obj:

  an **`xds`** model object

## Value

an **`xds`** object
