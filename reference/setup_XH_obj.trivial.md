# xde_setup XH_obj.trivial

Implements
[setup_XH_obj](https://dd-harp.github.io/ramp.xds/reference/setup_XH_obj.md)
for the trivial model

## Usage

``` r
# S3 method for class 'trivial'
setup_XH_obj(Xname, xds_obj, i, options = list())
```

## Arguments

- Xname:

  the model name

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

- options:

  model options as a named list

## Value

a [list](https://rdrr.io/r/base/list.html) vectord
