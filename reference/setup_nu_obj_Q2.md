# Setup Blood Feeding Bionomic Object

Set up an object to compute dynamic egg laying rates as a type 2
functional response to available habitat, \\Q\\

## Usage

``` r
setup_nu_obj_Q2(MY_obj, options = list(), nux = 0.35, snu = 1)
```

## Arguments

- MY_obj:

  an **`MY`** model object

- options:

  a list of options

- nux:

  the maximum egg laying rate

- snu:

  a shape parameter

## Value

a **`MY`** model object
