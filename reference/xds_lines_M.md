# Add lines for adult mosquito population density

Add lines for adult mosquito population density

## Usage

``` r
xds_lines_M(MY, xds_obj, clrs = "darkblue", llty = 1)
```

## Arguments

- MY:

  xds_objed ouptuts

- xds_obj:

  an **`xds`** model object

- clrs:

  a vector of colors

- llty:

  an integer (or integers) to set the `lty` for plotting
