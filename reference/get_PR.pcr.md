# Get the *Pf*PR from a MalariasModel

Return the *Pf*PR by PCR

## Usage

``` r
# S3 method for class 'pcr'
get_PR(xds_obj, i = 1, method = "pcr")
```

## Arguments

- xds_obj:

  an **`xds`** object

- i:

  the host species index

- method:

  the method used for computing *Pf*PR

## Value

none
