# Add indices for adult mosquitoes to parameter list

Implements
[setup_MY_ix](https://dd-harp.github.io/ramp.xds/reference/setup_MY_ix.md)
for the basic M model.

## Usage

``` r
# S3 method for class 'basicM'
setup_MY_ix(xds_obj, s)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the species index

## Value

none
