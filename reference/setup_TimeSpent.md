# Make a time spent matrix, called timespent

Make a time spent matrix, called timespent

## Usage

``` r
setup_timespent(name, xds_obj, i, options = list())
```

## Arguments

- name:

  a matrix or setup function name

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

- options:

  configuration options

## Value

an **`xds`** object
