# Set yy

Implement `change_spline` for a model with no forcing

## Usage

``` r
# S3 method for class 'none'
change_spline(X, xds_obj, s = 1)
```

## Arguments

- X:

  new interpolation points, as a list

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

a **`ramp.xds`** model object
