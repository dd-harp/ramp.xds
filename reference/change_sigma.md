# Change the mosquito patch emigration rate

Change the mosquito patch emigration rate for a static model

## Usage

``` r
change_sigma(sigma, xds_obj, s)
```

## Arguments

- sigma:

  the mosquito patch emigration rate

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

an **`xds`** model object
