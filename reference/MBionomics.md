# Mosquito bionomics (**MY**)

Compute the adult mosquito bionomic parameter values

## Usage

``` r
MBionomics(t, y, xds_obj, s)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

an **`xds`** object
