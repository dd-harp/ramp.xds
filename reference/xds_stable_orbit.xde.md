# Compute stable orbit for a system of differential equations

Run a system of differential equations for 10 years, and save the last
year's orbits.

## Usage

``` r
# S3 method for class 'xde'
xds_stable_orbit(xds_obj, options = list())
```

## Arguments

- xds_obj:

  an **`xds`** model object

- options:

  a list of configurable options

## Value

an **`xds`** object
