# Return the parameters as a list

This method dispatches on the type of `xds_obj$MY_obj[[s]]`.

## Usage

``` r
# S3 method for class 'macdonald'
get_MY_pars(xds_obj, s = 1)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

a [list](https://rdrr.io/r/base/list.html)
