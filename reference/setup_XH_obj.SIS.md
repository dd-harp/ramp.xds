# Setup `SIS` (**XH** component)

Set up an `SIS` model object for the **XH** component

## Usage

``` r
# S3 method for class 'SIS'
setup_XH_obj(Xname, xds_obj, i, options = list())
```

## Arguments

- Xname:

  the model name

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

- options:

  model options as a named list

## Value

the **`xds`** model object
