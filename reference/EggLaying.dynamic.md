# Compute eggs laid

Computes eggs laid with exogenous forcing on parameter affecting host
availability.

## Usage

``` r
# S3 method for class 'dynamic'
EggLaying(t, y, xds_obj)
```

## Arguments

- t:

  the time

- y:

  the state variables

- xds_obj:

  an **`xds`** model object

## Value

an `xds` object
