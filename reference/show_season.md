# Plot the seasonal pattern

For a model forced by emergence, show the seasonal pattern

## Usage

``` r
show_season(xds_obj, tm = seq(0, 730, by = 5), add = FALSE)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- tm:

  the time points

- add:

  add to existing plot

## Value

the seasonal pattern, invisibly
