# Get Initial Values for the **L** Component

Get Initial Values for the **L** Component

## Usage

``` r
get_L_inits(xds_obj, s = 1)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the species index

## Value

a named [list](https://rdrr.io/r/base/list.html)

## Note

This method dispatches on the class of `xds_obj$L_obj`.
