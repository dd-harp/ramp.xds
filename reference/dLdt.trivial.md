# Derivatives for the `trivial` **L** module

Returns a numeric vector of length 0

## Usage

``` r
# S3 method for class 'trivial'
dLdt(t, y, xds_obj, s)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

- s:

  the species index

## Value

an empty [list](https://rdrr.io/r/base/list.html)
