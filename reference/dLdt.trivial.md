# Compute derivatives for `trivial` (**L**)

Returns a numeric vector of length 0

## Usage

``` r
# S3 method for class 'trivial'
dLdt(t, y, xds_obj, s)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

- s:

  the species index

## Value

a [numeric](https://rdrr.io/r/base/numeric.html) vector of length 0
