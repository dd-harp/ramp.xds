# Compute the stable orbit for a discrete time system

Run a discrete time system for 10 years, and save the last year's
orbits.

## Usage

``` r
# S3 method for class 'dts'
xds_stable_orbit(xds_obj, options = list())
```

## Arguments

- xds_obj:

  an **`xds`** model object

- options:

  a list of configurable options

## Value

an **`xds`** object
