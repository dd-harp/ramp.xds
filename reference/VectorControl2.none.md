# Set the values of exogenous variables

Do no vector control: the **`xds`** object is returned unmodified

## Usage

``` r
# S3 method for class 'none'
VectorControl2(t, y, xds_obj)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

## Value

a **`ramp.xds`** model object
