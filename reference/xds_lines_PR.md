# Add lines for the prevalence / parasite rate (PR) from a model of human infection and immunity

Add lines for the prevalence / parasite rate (PR) from a model of human
infection and immunity

## Usage

``` r
xds_lines_PR(xds_obj, i, clrs = "black", llty = 1)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

- clrs:

  a vector of colors

- llty:

  an integer (or integers) that specifies `lty` for plotting

## Value

true pr, invisibly
