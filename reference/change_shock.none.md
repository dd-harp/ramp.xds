# Set yy

Implement `change_shock` for a model with no forcing

## Usage

``` r
# S3 method for class 'none'
change_shock(shock_par, xds_obj, s = 1)
```

## Arguments

- shock_par:

  parameters for
  [make_function](https://dd-harp.github.io/ramp.xds/reference/make_function.md)

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

a **`ramp.xds`** model object
