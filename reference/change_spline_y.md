# Set yy

Set new \\y\\-values for the spline interpolating points and update
`F_trend`

## Usage

``` r
change_spline_y(yy, xds_obj, s = 1)
```

## Arguments

- yy:

  new y values for the interpolation points

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

a **`ramp.xds`** model object
