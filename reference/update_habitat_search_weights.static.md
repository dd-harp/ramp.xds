# Update habitat search weights (static)

Returns `xds_obj` unmodified; search weights are static.

## Usage

``` r
# S3 method for class 'static'
update_habitat_search_weights(xds_obj, s)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

an **`xds`** object
