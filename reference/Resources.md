# Resources

Resources is a junction that optionally updates parameters affecting
adult mosquito bionomics: blood hosts, aquatic habitat dynamics, traps,
and sugar.

## Usage

``` r
Resources(t, y, xds_obj)
```

## Arguments

- t:

  the time

- y:

  the state variables

- xds_obj:

  an **`xds`** model object

## Value

an **`xds`** object
