# Plot the fraction of infected and infective mosquitoes

Plot the fraction of infected and infective mosquitoes

## Usage

``` r
xds_plot_Y_fracs(xds_obj, s = 1, clrs = "purple", llty = 1, add = FALSE)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

- clrs:

  a vector of colors for infected mosquitoes

- llty:

  an integer (or integers) to set the `lty` for plotting

- add:

  do not plot axes if TRUE
