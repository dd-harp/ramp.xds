# Set mean forcing

Implement `change_mean_forcing` for a model with no forcing

## Usage

``` r
# S3 method for class 'none'
change_mean_forcing(X, xds_obj, s = 1)
```

## Arguments

- X:

  the new mean forcing parameter

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

a **`ramp.xds`** model object
