# Add lines for the EIR *vs.* time

Add lines for the EIR *vs.* time

## Usage

``` r
xds_lines_EIR(xds_obj, i = 1, clrs = "black", llty = 1, annual = TRUE)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

- clrs:

  a vector of colors

- llty:

  an integer (or integers) to set the `lty` for plotting

- annual:

  if true, plot as an annualized rate

## Value

eir, invisibly
