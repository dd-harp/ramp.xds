# Update Blood Search Weights (static)

Returns `xds_obj` unmodified; search weights are static.

## Usage

``` r
# S3 method for class 'static'
update_blood_search_weights(xds_obj, s, i)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

- i:

  the host species index

## Value

an **`xds`** object
