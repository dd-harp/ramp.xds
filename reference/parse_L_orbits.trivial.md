# Parse outputs for `trivial` (**L**)

Returns an empty list

## Usage

``` r
# S3 method for class 'trivial'
parse_L_orbits(outputs, xds_obj, s)
```

## Arguments

- outputs:

  a [matrix](https://rdrr.io/r/base/matrix.html) with the solutions to a
  dynamical system

- xds_obj:

  an **`xds`** model object

- s:

  the species index

## Value

an empty [list](https://rdrr.io/r/base/list.html)
