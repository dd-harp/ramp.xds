# Steady States for **X**

Compute the steady states as a function of the daily FoI for a static
value of human population density

## Usage

``` r
steady_state_X(foi, H, xds_obj, i = 1)
```

## Arguments

- foi:

  the daily FoI

- H:

  human / host population density

- xds_obj:

  an **`xds`** model object

- i:

  the vector species index

## Value

steady states
