# Change habitat search weights

Change the habitat feeding search weights, \\\omega\\, for a set of host
strata

## Usage

``` r
change_habitat_search_weights(wts, xds_obj, s = 1)
```

## Arguments

- wts:

  the habitat feeding search weights

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

an **`xds`** object
