# Static model for the blood feeding rate

Implements
[F_maturation](https://dd-harp.github.io/ramp.xds/reference/F_maturation.md)
for a static model

## Usage

``` r
# S3 method for class 'static'
F_maturation(t, xds_obj, s)
```

## Arguments

- t:

  current simulation time

- xds_obj:

  an **`xds`** model object

- s:

  vector species index

## Value

\\psi\\, the baseline human fraction
