# View residence membership

Shows the residence membership information (from the residence matrix)

## Usage

``` r
view_residency_matrix(xds_obj, i = 1)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

## Value

a named [list](https://rdrr.io/r/base/list.html)
