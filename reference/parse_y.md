# parse the output of an object returned by deSolve

parse the output of an object returned by deSolve

## Usage

``` r
parse_y(y, xds_obj)
```

## Arguments

- y:

  a [vector](https://rdrr.io/r/base/vector.html) with the variables

- xds_obj:

  an **`xds`** model object

## Value

varslist a [list](https://rdrr.io/r/base/list.html)
