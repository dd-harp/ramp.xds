# Check the shape and dimensions of an model object

Check the shape and dimensions of an model object

## Usage

``` r
shapeIt(obj, d1, d2)
```

## Arguments

- obj:

  a [numeric](https://rdrr.io/r/base/numeric.html) model object

- d1:

  an [integer](https://rdrr.io/r/base/integer.html)

- d2:

  an [integer](https://rdrr.io/r/base/integer.html)

## Value

[matrix](https://rdrr.io/r/base/matrix.html)
