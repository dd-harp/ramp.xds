# Return the parameters as a list

Return the parameters as a list

## Usage

``` r
# S3 method for class 'SIS'
change_XH_inits(xds_obj, i = 1, options = list())
```

## Arguments

- xds_obj:

  an **`xds`** model object

- i:

  the vector species index

- options:

  a named list

## Value

an **`xds`** object
