# Get the density of infectious, blood feeding mosquitoes

Get the density of infectious, blood feeding mosquitoes

## Usage

``` r
get_Z(xds_obj, s = 1)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

y a [numeric](https://rdrr.io/r/base/numeric.html) vector assigned the
class "dynamic"
