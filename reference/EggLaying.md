# Compute eggs laid

Compute eggs laid at time \\t\\

## Usage

``` r
EggLaying(t, y, xds_obj)
```

## Arguments

- t:

  the time

- y:

  the state variables

- xds_obj:

  an **`xds`** model object

## Value

an **`xds`** model object
