# The **XH** Module Skill Set

The **XH** skill set is a list of an module's capabilities. The relevant
documentation for setup is in
[make_XH_obj_trivial](https://dd-harp.github.io/ramp.xds/reference/make_XH_obj_trivial.md)

## Usage

``` r
# S3 method for class 'trivial'
skill_set_XH(Xname)
```

## Arguments

- Xname:

  the **XH** module name

## Value

the trivial *XH* module skill set, a list
