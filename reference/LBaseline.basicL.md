# Baseline Bionomics for `basicL` (**L** Component)

Set **L** component parameters to baseline values for `basicL`

## Usage

``` r
# S3 method for class 'basicL'
LBaseline(t, y, xds_obj, s)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

- s:

  the species index

## Value

a **`ramp.xds`** object
