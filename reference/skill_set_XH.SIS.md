# The **XH** Module Skill Set

The **XH** skill set is a list of an module's capabilities.

## Usage

``` r
# S3 method for class 'SIS'
skill_set_XH(Xname = "SIS")
```

## Arguments

- Xname:

  the **XH** module name

## Value

the skill set, as a list

## Note

This method dispatches on `class(xds_obj$XH_obj)`
