# Change Habitat Search Weights

Set the search weights, \\\omega\\, for a set of aquatic habitats

## Usage

``` r
change_habitat_weights(wts, xds_obj, s = 1)
```

## Arguments

- wts:

  habitat search weights

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

an **`xds`** model object
