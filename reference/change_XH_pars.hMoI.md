# Return the parameters as a list

This method dispatches on the type of `xds_obj$XH_obj[[i]]`.

## Usage

``` r
# S3 method for class 'hMoI'
change_XH_pars(xds_obj, i = 1, options = list())
```

## Arguments

- xds_obj:

  an **`xds`** model object

- i:

  the vector species index

- options:

  a named list

## Value

an **`xds`** object
