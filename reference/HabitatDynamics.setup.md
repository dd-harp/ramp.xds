# Habitat Dynamics

This function modifies search weights for habitats and availability of
bad habitat

## Usage

``` r
# S3 method for class 'setup'
HabitatDynamics(t, y, xds_obj)
```

## Arguments

- t:

  current simulation time

- y:

  state variables

- xds_obj:

  an **`xds`** model object

## Value

an **`xds`** object
