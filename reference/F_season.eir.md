# Compute the seasonal pattern

Evaluate the function `F_season` for a forced model when `forced_by` is
"eir"

## Usage

``` r
# S3 method for class 'eir'
F_season(tm, xds_obj)
```

## Arguments

- tm:

  the time points

- xds_obj:

  an **`xds`** model object

## Value

the seasonal pattern, invisibly
