# Setup Bad Habitat Availability

Setup a the function model for no time spent searching and no exposure
while searching

## Usage

``` r
# S3 method for class 'static'
setup_F_bad_habitat(mod_name, xds_obj, s, options = list())
```

## Arguments

- mod_name:

  the model name

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

- options:

  a list of setup options (see
  [make_ts_function](https://dd-harp.github.io/ramp.xds/reference/make_ts_function.md))

## Value

an **`xds`** model object
