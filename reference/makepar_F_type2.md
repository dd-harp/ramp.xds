# parameters for make_function

Return an object to configure a function
[make_function.type2](https://dd-harp.github.io/ramp.xds/reference/make_function.type2.md)

## Usage

``` r
makepar_F_type2(shift = 30, A = 1.8, B = 5, N = 1)
```

## Arguments

- shift:

  a shift to set F(0)\>0

- A:

  asymptote

- B:

  slope

- N:

  the length of the vector to return

## Value

a type 2 functional response
