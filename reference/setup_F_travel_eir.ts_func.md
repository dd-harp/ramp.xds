# Setup the Travel EIR

Setup a model for no time spent traveling and no exposure while
traveling

## Usage

``` r
# S3 method for class 'ts_func'
setup_F_travel_eir(setup_name, xds_obj, i, options)
```

## Arguments

- setup_name:

  a

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

- options:

  options list to set up `F_travel_eir`

## Value

an **`xds`** model object
