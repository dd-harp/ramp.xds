# Time Spent F_travel

For a static model for time spent F_travel, the function does not update
anything.

## Usage

``` r
# S3 method for class 'static'
Travel(t, y, xds_obj)
```

## Arguments

- t:

  current simulation time

- y:

  variables

- xds_obj:

  an **`xds`** model object

## Value

an **`xds`** object
