# Get the *Pf*PR from a Malaria Model

Return the PR by light microscopy

## Usage

``` r
# S3 method for class 'lm'
get_PR(xds_obj, i = 1, method = "lm")
```

## Arguments

- xds_obj:

  an **`xds`** object

- i:

  the host species index

- method:

  the method used for computing *Pf*PR

## Value

none
