# Compute time spent objects: static models

Return the time spent objects unmodified

## Usage

``` r
# S3 method for class 'static'
TimeSpent(t, y, xds_obj)
```

## Arguments

- t:

  the time

- y:

  the state variables

- xds_obj:

  an **`xds`** model object

## Value

the unmodified `xds` object
