# Compute Infectious Density

In the *SIS* model family, infectious density is \\cI\\.

## Usage

``` r
# S3 method for class 'SIS'
F_I(t, y, xds_obj, i)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

## Value

Infectious density
