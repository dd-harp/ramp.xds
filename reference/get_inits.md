# Get the stored initial values, \\y_0\\

Get the stored initial values, \\y_0\\

## Usage

``` r
get_inits(xds_obj, flatten = FALSE)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- flatten:

  a [logical](https://rdrr.io/r/base/logical.html): if true, results are
  returned as an unnamed vector

## Value

a named [list](https://rdrr.io/r/base/list.html) or `if(flatten==TRUE)`
a [vector](https://rdrr.io/r/base/vector.html)
