# Exogenous Forcing

After basic setup, no exogenous variables are set up. `Forcing` returns
the unmodified **`xds`** object.

## Usage

``` r
# S3 method for class 'none'
Forcing(t, xds_obj)
```

## Arguments

- t:

  current simulation time

- xds_obj:

  an **`xds`** model object

## Value

an **`xds`** object
