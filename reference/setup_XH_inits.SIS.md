# Setup initial values for *SIS*

Setup initial values for *SIS*

## Usage

``` r
# S3 method for class 'SIS'
setup_XH_inits(xds_obj, H, i = 1, options = list())
```

## Arguments

- xds_obj:

  an **`xds`** model object

- H:

  initial host population density

- i:

  the host species index

- options:

  a [list](https://rdrr.io/r/base/list.html)

## Value

a **`ramp.xds`** object
