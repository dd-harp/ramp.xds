# Get VC

Get the VC matrix

## Usage

``` r
get_VC_matrix(xds_obj, s = 1, computed = TRUE)
```

## Arguments

- xds_obj:

  a **`ramp.xds`** model object

- s:

  the vector species index

- computed:

  if true, get the VC matrix that was computed; if false, by formula

## Value

a numeric [matrix](https://rdrr.io/r/base/matrix.html)
