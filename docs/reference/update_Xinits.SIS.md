# Update inits for the SIS xde human model from a vector of states

Update inits for the SIS xde human model from a vector of states

## Usage

``` r
# S3 method for SIS
update_Xinits(pars, y, i = 1)
```

## Arguments

- pars:

  an **`xds`** object

- y:

  the variables

- i:

  the host species index

## Value

an `xds` object
