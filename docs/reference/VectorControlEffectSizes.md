# Vector control effect sizes

This method dispatches on the type of `xds_obj$VECTOR_CONTROL`.

## Usage

``` r
VectorControlEffectSizes(t, y, xds_obj)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

## Value

a **`ramp.xds`** model object
