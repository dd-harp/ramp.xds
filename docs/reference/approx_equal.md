# Check if two numeric values are approximately equal

Check if two numeric values are approximately equal

## Usage

``` r
approx_equal(a, b, tol = sqrt(.Machine$double.eps))
```

## Arguments

- a:

  a [numeric](https://rdrr.io/r/base/numeric.html) model object

- b:

  a [numeric](https://rdrr.io/r/base/numeric.html) model object

- tol:

  the numeric tolerance

## Value

a [logical](https://rdrr.io/r/base/logical.html) value
