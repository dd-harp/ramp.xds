# Set up Mosquito traps

Create a junction for mosquito traps

## Usage

``` r
Traps(t, y, xds_obj)
```

## Arguments

- t:

  current simulation time

- y:

  state variables

- xds_obj:

  an **`xds`** model object

## Value

an **`xds`** object

## Note

Mosquito trap models are in `ramp.control`
