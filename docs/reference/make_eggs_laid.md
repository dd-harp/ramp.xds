# Compute eggs laid

Computes eggs laid in aquatic habitats

## Usage

``` r
make_eggs_laid(t, y, pars)
```

## Arguments

- t:

  the time

- y:

  the state variables

- pars:

  an `xds` object

## Value

an `xds` object
