# Update Blood Search Weights (setup)

Acknowledges a one-time update to search weights and sets the port back
to `"static"`.

## Usage

``` r
# S3 method for class 'setup'
update_search_weights(xds_obj, s, i)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

- i:

  the host species index

## Value

an **`xds`** object
