# A run-time switch function for mismatched dynamical component run-times

Determine whether to update the variables at time t

## Usage

``` r
runt(t, Dday, xday)
```

## Arguments

- t:

  current simulation time

- Dday:

  the run-time time step for the simulation

- xday:

  the run-time time step for a component

## Value

[logical](https://rdrr.io/r/base/logical.html) TRUE or FALSE
