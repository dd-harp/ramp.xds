# Solve a system of equations as an ode

Implements for ordinary differential equations

## Usage

``` r
xde_cohort_desolve(birthday, inits, times, xds_obj)
```

## Arguments

- birthday:

  a cohort birthday

- inits:

  initial values

- times:

  = the times

- xds_obj:

  an **`xds`** model object

## Value

a [list](https://rdrr.io/r/base/list.html)
