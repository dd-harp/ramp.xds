# Set yy

Set the yy for the seasonal pattern for an `eir` model and return the
**`ramp.xds`** model object

## Usage

``` r
# S3 method for class 'eir'
change_shock(shock_par, xds_obj, s = 1)
```

## Arguments

- shock_par:

  parameters for
  [make_function](https://dd-harp.github.io/ramp.xds/reference/make_function.md)

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

a **`ramp.xds`** model object
