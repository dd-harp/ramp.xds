# Setup the basicM model

Implements
[setup_MY_inits](https://dd-harp.github.io/ramp.xds/reference/setup_MY_inits.md)
for the basicM model

## Usage

``` r
# S3 method for class 'basicM'
setup_MY_inits(xds_obj, s, options)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the species index

- options:

  a [list](https://rdrr.io/r/base/list.html)

## Value

a [list](https://rdrr.io/r/base/list.html) vector
