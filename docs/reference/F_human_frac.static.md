# Static model for the blood feeding rate

Implements
[F_human_frac](https://dd-harp.github.io/ramp.xds/reference/F_human_frac.md)
for a static model

## Usage

``` r
# S3 method for class 'static'
F_human_frac(t, xds_obj, s)
```

## Arguments

- t:

  current simulation time

- xds_obj:

  an **`xds`** model object

- s:

  vector species index

## Value

\\q\\, the baseline human fraction
