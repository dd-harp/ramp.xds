# Infection blocking pre-erythrocytic immunity

This method dispatches on the type of `pars$Xpar[[i]]`.

## Usage

``` r
F_b(y, pars, i)
```

## Arguments

- y:

  state vector

- pars:

  an **`xds`** object

- i:

  the host species index

## Value

a [numeric](https://rdrr.io/r/base/numeric.html) vector of length
`nStrata`
