# Compute the steady states for the dts SIS model as a function of the daily EIR

Compute the steady state of the dts SIS model as a function of the daily
eir.

## Usage

``` r
# S3 method for SIS
dts_steady_state_X(ar, H, Xpar)
```

## Arguments

- ar:

  the daily attack rate

- H:

  host density

- Xpar:

  a list that defines a X-class model

## Value

the steady states as a named vector
