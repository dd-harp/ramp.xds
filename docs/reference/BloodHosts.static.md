# Time Spent F_blood_hosts

For a static model for time spent F_blood_hosts, the function does not
update anything.

## Usage

``` r
# S3 method for class 'static'
BloodHosts(t, y, xds_obj)
```

## Arguments

- t:

  current time

- y:

  the state variable vector

- xds_obj:

  an **`xds`** model object

## Value

an **`xds`** object
