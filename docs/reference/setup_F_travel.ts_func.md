# Set up no travel

Setup a model for no time spent traveling and no exposure while
traveling

## Usage

``` r
# S3 method for class 'ts_func'
setup_F_travel(setup_name, xds_obj, i, options = list())
```

## Arguments

- setup_name:

  a string to dispatch `setup_F_travel`

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

- options:

  setup options for `F_travel`

## Value

an **`xds`** model object
