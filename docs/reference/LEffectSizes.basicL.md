# Bionomics for `basicL` (**L** Component)

Implements
[LEffectSizes](https://dd-harp.github.io/ramp.xds/reference/LEffectSizes.md)
for the `basicL`

## Usage

``` r
# S3 method for class 'basicL'
LEffectSizes(t, y, xds_obj, s)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

- s:

  the species index

## Value

an **`xds`** object
