# Travel Malaria

Travel malaria is defined as exposure to malaria acquired outside of the
spatial domain represented by the patches. The user can configure two
ports:

- time spent away / here: see
  [xds_port_time_spent_here](https://dd-harp.github.io/ramp.xds/reference/xds_port_time_spent_here.md)

- the travel EIR: see
  [xds_port_travel_eir](https://dd-harp.github.io/ramp.xds/reference/xds_port_travel_eir.md)

Travel malaria is implemented in the **Exposure**
