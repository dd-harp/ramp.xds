# Set up Indices for `trivial` (**L**)

Return the **`xds`** object unmodified

## Usage

``` r
# S3 method for class 'trivial'
setup_L_ix(xds_obj, s)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the species index

## Value

an **`xds`** object
