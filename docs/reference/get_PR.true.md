# Get the *Pf*PR from a Malaria Model

Return the true *Pf*PR

## Usage

``` r
# S3 method for class 'true'
get_PR(xds_obj, method = "true", i = 1)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- method:

  the method used for computing *Pf*PR

- i:

  the host species index

## Value

the true PR
