# Get **L** Component Parameters for `basicL`

Get the **L** component parameters

## Usage

``` r
# S3 method for class 'basicL'
get_L_pars(xds_obj, s = 1)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

a [list](https://rdrr.io/r/base/list.html)
