# Get Variables by Name

Get the the values of variables from the flat state variable vector
\\y\\, and return the values as a named list

## Usage

``` r
get_V_vars(y, xds_obj, i = 1)
```

## Arguments

- y:

  the variables

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

## Value

Variables as a named list
