# Update the time spent Matrix (static)

Returns `xds_obj` unmodified; the time spent matrix is static.

## Usage

``` r
# S3 method for class 'static'
update_timespent(xds_obj, i)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

## Value

an **`xds`** object
