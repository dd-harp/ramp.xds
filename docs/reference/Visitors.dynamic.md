# Visitor Dynamics

This function sets the value of a parameter describing the availability
of visitors.

## Usage

``` r
# S3 method for class 'dynamic'
Visitors(t, y, xds_obj)
```

## Arguments

- t:

  current time

- y:

  the state variable vector

- xds_obj:

  an **`xds`** model object

## Value

an **`xds`** object
