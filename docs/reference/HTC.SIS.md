# Compute the HTC for the SIS model

Implements [HTC](https://dd-harp.github.io/ramp.xds/reference/HTC.md)
for the SIS model with demography.

## Usage

``` r
# S3 method for class 'SIS'
HTC(xds_obj, i)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

## Value

a [numeric](https://rdrr.io/r/base/numeric.html) vector
