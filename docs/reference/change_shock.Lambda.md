# Set the \\y\\ value for interpolating points

and return the **`ramp.xds`** model object

## Usage

``` r
# S3 method for class 'Lambda'
change_shock(shock_par, xds_obj, s = 1)
```

## Arguments

- shock_par:

  parameters for
  [make_function](https://dd-harp.github.io/ramp.xds/reference/make_function.md)

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

the **`ramp.xds`** model object
