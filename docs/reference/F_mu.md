# Compute the emigration loss

Compute the fraction lost to emigration

## Usage

``` r
F_mu(t, vars, mu_par)
```

## Arguments

- t:

  current simulation time

- vars:

  exogenous variables

- mu_par:

  a [list](https://rdrr.io/r/base/list.html)

## Value

a [numeric](https://rdrr.io/r/base/numeric.html) vector of length
`nPatches`
