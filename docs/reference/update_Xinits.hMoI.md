# Update inits for hybrid MoI human model from a vector of states

Update inits for hybrid MoI human model from a vector of states

## Usage

``` r
# S3 method for hMoI
update_Xinits(pars, y, i)
```

## Arguments

- pars:

  an **`xds`** object

- y:

  the variables

- i:

  the host species index

## Value

none
