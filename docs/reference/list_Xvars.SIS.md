# Return the variables as a list

This method dispatches on the type of `pars$Xpar`

## Usage

``` r
# S3 method for SIS
list_Xvars(y, pars, i)
```

## Arguments

- y:

  the variables

- pars:

  an **`xds`** object

- i:

  the host species index

## Value

a [list](https://rdrr.io/r/base/list.html)
