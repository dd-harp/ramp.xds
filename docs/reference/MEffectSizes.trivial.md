# Apply effect sizes for `trivial` (**MY**)

Implements
[MEffectSizes](https://dd-harp.github.io/ramp.xds/reference/MEffectSizes.md)
for the `trivial` module

## Usage

``` r
# S3 method for class 'trivial'
MEffectSizes(t, y, xds_obj, s)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

an **`xds`** object
