# Update **L** Component Initial Values

This sets initial values for an **L** Component module from a state
variables vector \\y\\

## Usage

``` r
update_Linits(pars, y, s)
```

## Arguments

- pars:

  an **`xds`** object

- y:

  the state variables

- s:

  the species index

## Value

an `xds` object
