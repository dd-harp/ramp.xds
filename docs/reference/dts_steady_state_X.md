# Compute the steady states as a function of the daily EIR

This method dispatches on the type of `pars$Xpar[[i]]`

## Usage

``` r
dts_steady_state_X(ar, H, Xpar)
```

## Arguments

- ar:

  the daily attack rate

- H:

  host density

- Xpar:

  a list that defines a X-class model

## Value

none
