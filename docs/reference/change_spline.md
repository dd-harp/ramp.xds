# Set the interpolating points

Set the interpolating points for F_trend

## Usage

``` r
change_spline(X, xds_obj, s = 1)
```

## Arguments

- X:

  new interpolation points, as a list

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

a **`ramp.xds`** model object
