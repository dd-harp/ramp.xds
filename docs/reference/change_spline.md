# Change spline interpolation points

Change the interpolation points for a spline function.

## Usage

``` r
change_spline(X, xds_obj, s = 1)
```

## Arguments

- X:

  new interpolation points, as a list

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

an **`xds`** object
