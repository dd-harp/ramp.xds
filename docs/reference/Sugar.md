# Set up Mosquito sugar

Create a junction for mosquito sugar

## Usage

``` r
Sugar(t, y, xds_obj)
```

## Arguments

- t:

  current simulation time

- y:

  dependent variables vector

- xds_obj:

  an **`xds`** model object

## Value

an **`xds`** object

## Note

Mosquito trap models are in `ramp.control`
