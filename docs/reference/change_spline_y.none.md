# Set yy

Implement `change_spline_y` for a model with no forcing

## Usage

``` r
# S3 method for class 'none'
change_spline_y(yy, xds_obj, s = 1)
```

## Arguments

- yy:

  new y values for the interpolation points

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

a **`ramp.xds`** model object
