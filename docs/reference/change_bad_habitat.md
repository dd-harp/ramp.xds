# Change Habitat Search Weights

Set the search weights, \\\omega\\, for a set of aquatic habitats

## Usage

``` r
change_bad_habitat(Qbad, xds_obj, s = 1)
```

## Arguments

- Qbad:

  availability of bad habitat

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

an `xds` object
