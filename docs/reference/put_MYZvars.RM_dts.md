# Return the variables as a list

This method dispatches on the type of `pars$MYZpar[[s]]`

## Usage

``` r
# S3 method for RM_dts
put_MYZvars(MYZvars, y, pars, s)
```

## Arguments

- MYZvars:

  the variables

- y:

  the variables

- pars:

  a [list](https://rdrr.io/r/base/list.html)

- s:

  the vector species index

## Value

a [list](https://rdrr.io/r/base/list.html)
