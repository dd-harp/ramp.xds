# Mosquito bionomics (**L**)

Compute the values of bionomic parameter values for an **L** Component
module

## Usage

``` r
LBionomics(t, y, xds_obj, s)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

- s:

  the species index

## Value

an **`xds`** object
