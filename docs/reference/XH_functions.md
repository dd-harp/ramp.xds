# Generic methods for the XH component

The functions that define **XH**:

- [`dXHdt`](https://dd-harp.github.io/ramp.xds/reference/dXHdt.md) –
  compute derivatives

- [`Update_XHt`](https://dd-harp.github.io/ramp.xds/reference/Update_XHt.md)
  – update state variables

- [`F_H`](https://dd-harp.github.io/ramp.xds/reference/F_H.md) – compute
  host population density

- [`F_I`](https://dd-harp.github.io/ramp.xds/reference/F_I.md) – compute
  infectious host population size

- [`F_infectivity`](https://dd-harp.github.io/ramp.xds/reference/F_infectivity.md)
  – compute infectivity (pre-erythrocytic immunity)

- [`F_ni`](https://dd-harp.github.io/ramp.xds/reference/F_ni.md) –
  compute net infectiousness

- [`F_prevalence`](https://dd-harp.github.io/ramp.xds/reference/F_prevalence.md)
  – compute true prevalence

- [`F_pfpr_by_lm`](https://dd-harp.github.io/ramp.xds/reference/F_pfpr_by_lm.md)
  – compute prevalence by light microscopy

- [`F_pfpr_by_rdt`](https://dd-harp.github.io/ramp.xds/reference/F_pfpr_by_rdt.md)
  – compute prevalence by RDT

- [`F_pfpr_by_pcr`](https://dd-harp.github.io/ramp.xds/reference/F_pfpr_by_pcr.md)
  – compute prevalence by PCR

- [`get_HTC`](https://dd-harp.github.io/ramp.xds/reference/get_HTC.md) –
  compute human transmitting capacity

- [`setup_XH_obj`](https://dd-harp.github.io/ramp.xds/reference/setup_XH_obj.md)
  – set up the XH module object

- [`setup_XH_ix`](https://dd-harp.github.io/ramp.xds/reference/setup_XH_ix.md)
  – set index values

- [`setup_XH_inits`](https://dd-harp.github.io/ramp.xds/reference/setup_XH_inits.md)
  – set up initial values

- [`get_XH_inits`](https://dd-harp.github.io/ramp.xds/reference/get_XH_inits.md)
  – get initial values

- [`change_XH_inits`](https://dd-harp.github.io/ramp.xds/reference/change_XH_inits.md)
  – change initial values

- [`get_XH_vars`](https://dd-harp.github.io/ramp.xds/reference/get_XH_vars.md)
  – list state variables

- [`get_XH_ix`](https://dd-harp.github.io/ramp.xds/reference/get_XH_ix.md)
  – get variable indices

- [`get_XH_pars`](https://dd-harp.github.io/ramp.xds/reference/get_XH_pars.md)
  – get parameters

- [`change_XH_pars`](https://dd-harp.github.io/ramp.xds/reference/change_XH_pars.md)
  – change parameters

- [`change_H`](https://dd-harp.github.io/ramp.xds/reference/change_H.md)
  – change host population density

- [`parse_XH_orbits`](https://dd-harp.github.io/ramp.xds/reference/parse_XH_orbits.md)
  – parse outputs

- [`get_XH_orbits`](https://dd-harp.github.io/ramp.xds/reference/get_XH_orbits.md)
  – get saved orbits

- [`get_PR`](https://dd-harp.github.io/ramp.xds/reference/get_PR.md) –
  get prevalence from orbits

- [`steady_state_X`](https://dd-harp.github.io/ramp.xds/reference/steady_state_X.md)
  – compute X steady state given FoI and H

- [`steady_state_XH`](https://dd-harp.github.io/ramp.xds/reference/steady_state_XH.md)
  – compute XH steady state given FoI

- [`xds_plot_X`](https://dd-harp.github.io/ramp.xds/reference/xds_plot_X.md)
  – plot X outputs

- [`check_XH`](https://dd-harp.github.io/ramp.xds/reference/check_XH.md)
  – run consistency checks

- [`skill_set_XH`](https://dd-harp.github.io/ramp.xds/reference/skill_set_XH.md)
  – return module skill set
