# Bionomics for `trivial` (**L** Component)

Implements
[LBionomics](https://dd-harp.github.io/ramp.xds/reference/LBionomics.md)
for the RM model

## Usage

``` r
# S3 method for class 'trivial'
LBionomics(t, y, xds_obj, s)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

- s:

  the species index

## Value

an **`xds`** object
