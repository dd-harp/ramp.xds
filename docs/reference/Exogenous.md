# Compute Derivatives

The function to compute the derivatives dispatches on `class(frame)`

## Usage

``` r
Exogenous(t, y, pars)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- pars:

  an **`xds`** object

## Value

an **`xds`** object
