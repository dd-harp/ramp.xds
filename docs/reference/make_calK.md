# Make a mosquito dispersal matrix, called calK

Make a mosquito dispersal matrix, called calK

## Usage

``` r
make_calK(calK, s, pars, opts = list())
```

## Arguments

- calK:

  a matrix or a string

- s:

  the vector species index

- pars:

  an `xds` object

- opts:

  a list of options to configure calK

## Value

a modified `xds` object
