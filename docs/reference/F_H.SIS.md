# Compute Population Density

Implements [F_H](https://dd-harp.github.io/ramp.xds/reference/F_H.md)
for the *SIS* module

## Usage

``` r
# S3 method for class 'SIS'
F_H(t, y, xds_obj, i)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

## Value

Human (host) population density, \\H\\
