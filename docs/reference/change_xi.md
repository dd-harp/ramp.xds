# Change xi, the maturation delay parameter

Change the maturation delay parameter for basicL for a static model

## Usage

``` r
change_xi(xi, xds_obj, s)
```

## Arguments

- xi:

  the density independent mortality rate

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

an **`xds`** model object
