# Set up a fixedlag_dts model for the EIP

Set up a fixedlag_dts model for the EIP

## Usage

``` r
setup_eip_fixedlag_dts(EIPopts, MYZpar, eip = 12, Dday = 1, MYZday = 1)
```

## Arguments

- EIPopts:

  a [list](https://rdrr.io/r/base/list.html)

- MYZpar:

  the MYZ parameters

- eip:

  a default value for the eip

- Dday:

  the simulation runtime interval

- MYZday:

  the MYZ component runtime interval

## Value

[list](https://rdrr.io/r/base/list.html)
