# Apply effect sizes (**L**)

Apply vector control effect sizes to immature mosquito bionomic
parameters

## Usage

``` r
LEffectSizes(t, y, xds_obj, s)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

- s:

  the species index

## Value

an **`xds`** object
