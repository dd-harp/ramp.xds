# Setup an **V** Module (Human / Host Epidemiology & Demography)

Set the parameter values and configure a model for the **X** Component

## Usage

``` r
setup_V_obj(Xname, xds_obj, i, options = list())
```

## Arguments

- Xname:

  the model name

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

- options:

  model options as a named list

## Value

an **`xds`** object
