# Setup Function for No Vector Control (default)

This sets `class(vector_control) <- 'none'` to dispatch the no vector
control option.

## Usage

``` r
setup_no_vector_control(xds_obj)
```

## Arguments

- xds_obj:

  an **`xds`** object

## Value

a **`ramp.xds`** model object
