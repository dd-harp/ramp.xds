# Add lines for the fraction of infected and infective mosquitoes

Add lines for the fraction of infected and infective mosquitoes

## Usage

``` r
xds_lines_Z_fracs(MY, xds_obj, clrs = "darkred", llty = 1)
```

## Arguments

- MY:

  xds_objed outputs

- xds_obj:

  an **`xds`** model object

- clrs:

  a vector of colors for infective mosquitoes

- llty:

  an integer (or integers) to set the `lty` for plotting
