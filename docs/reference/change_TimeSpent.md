# Change the time spent matrix

Change the time spent matrix

## Usage

``` r
change_TimeSpent(TiSp, pars, i = 1)
```

## Arguments

- TiSp:

  a time spent [matrix](https://rdrr.io/r/base/matrix.html)

- pars:

  a [list](https://rdrr.io/r/base/list.html)

- i:

  the host species index

## Value

a [list](https://rdrr.io/r/base/list.html)
