# Compute the steady states as a function of the daily EIR

This method dispatches on the type of `MYZpar`.

## Usage

``` r
# S3 method for macdonald
xde_steady_state_MYZ(Lambda, kappa, MYZpar)
```

## Arguments

- Lambda:

  the daily emergence rate of adult mosquitoes

- kappa:

  net infectiousness

- MYZpar:

  a list that defines an adult model

## Value

none
