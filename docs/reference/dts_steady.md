# Compute the steady state of a discrete time system

Run a discrete time system for `Tx` time steps to get the steady state

## Usage

``` r
dts_steady(xds_obj, Tx = 1000)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- Tx:

  the number of steps to equilibrium

## Value

an **`xds`** object
