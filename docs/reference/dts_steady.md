# Solve for the steady state of a system of equations

This method dispatches on the type of `xds_obj$dts`

## Usage

``` r
dts_steady(xds_obj, Tx = 1000)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- Tx:

  the number of steps to equilibrium

## Value

a [list](https://rdrr.io/r/base/list.html)
