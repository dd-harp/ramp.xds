# Change mean forcing

Change `Lambda` for when `forced_by` = "Lambda"

## Usage

``` r
# S3 method for class 'Lambda'
change_mean_forcing(X, xds_obj, s = 1)
```

## Arguments

- X:

  the new mean forcing parameter

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

an **`xds`** object
