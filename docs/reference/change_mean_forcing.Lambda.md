# Set mean forcing

Set `Lambda` to `X` for the \\s^{th}\\ species and return the
**`ramp.xds`** model object

## Usage

``` r
# S3 method for class 'Lambda'
change_mean_forcing(X, xds_obj, s = 1)
```

## Arguments

- X:

  the new mean forcing parameter

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

the **`ramp.xds`** model object
