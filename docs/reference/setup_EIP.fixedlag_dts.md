# Set up a fixedlag_dts model for the EIP

Set up a fixedlag_dts model for the EIP

## Usage

``` r
# S3 method for fixedlag_dts
setup_EIP(EIPopts, MYZpar)
```

## Arguments

- EIPopts:

  a list with options to setup EIPmod; must include EIPmod\$EIPname

- MYZpar:

  the MYZ parameters

## Value

[list](https://rdrr.io/r/base/list.html)
