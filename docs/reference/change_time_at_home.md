# Change the travel EIR

Set a new *static* value for travel EIR

## Usage

``` r
change_time_at_home(time_at_home, xds_obj, i)
```

## Arguments

- time_at_home:

  time spent in spatial domain (*i.e.* in the patches)

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

## Value

an `xds` object
