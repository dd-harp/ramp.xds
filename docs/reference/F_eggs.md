# Number of eggs laid by adult mosquitoes

This method dispatches on the type of `xds_obj$MY_obj`.

## Usage

``` r
F_eggs(t, y, xds_obj, s)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

- s:

  the species index

## Value

a [numeric](https://rdrr.io/r/base/numeric.html) vector of length
`nPatches`
