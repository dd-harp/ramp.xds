# Compute transmission terms with a static mixing matrix

In the static case, \\\beta\\ is not updated

## Usage

``` r
# S3 method for class 'static'
Transmission(t, y, xds_obj)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

## Value

an `xds` object
