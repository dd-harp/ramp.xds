# dts_update\_ States for Discrete-Time Systems

dts_update_s the state variables

## Usage

``` r
dts_update(t, y, xds_obj)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

## Value

a [list](https://rdrr.io/r/base/list.html) containing the vector of all
state derivatives
