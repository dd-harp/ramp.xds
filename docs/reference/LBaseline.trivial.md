# Baseline Bionomics for `trivial` (**L** Component)

Implements
[LBaseline](https://dd-harp.github.io/ramp.xds/reference/LBaseline.md)
for the RM model

## Usage

``` r
# S3 method for class 'trivial'
LBaseline(t, y, xds_obj, s)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

- s:

  the species index

## Value

a named [list](https://rdrr.io/r/base/list.html)
