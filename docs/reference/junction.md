# Junctions

In **`ramp.xds,`** clusters of
[port](https://dd-harp.github.io/ramp.xds/reference/port.md)s are
bundled into a **junction.** A junction is a structural element, a
function that is designed to handle one or more
[port](https://dd-harp.github.io/ramp.xds/reference/port.md)s.

- [Other_State_Variables](https://dd-harp.github.io/ramp.xds/reference/Other_State_Variables.md)
  – a junction to add new state variables, effectively extending the
  dynamical system

- [Forcing](https://dd-harp.github.io/ramp.xds/reference/Forcing.md) – a
  junction to handle exogenous forcing by weather, *etc.*

- [Vector_Control](https://dd-harp.github.io/ramp.xds/reference/Vector_Control.md)
  – a junction to handle vector control
