# Get the adult mosquito mortality rate(s)

Get the adult mosquito mortality rate(s)

## Usage

``` r
get_g(xds_obj, s = 1)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

y a [numeric](https://rdrr.io/r/base/numeric.html) vector assigned the
class "dynamic"
