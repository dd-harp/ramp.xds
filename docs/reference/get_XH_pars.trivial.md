# Return the parameters as a list

This method dispatches on the type of `xds_obj$XH_obj[[s]]`.

## Usage

``` r
# S3 method for class 'trivial'
get_XH_pars(xds_obj, i = 1)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

## Value

a [list](https://rdrr.io/r/base/list.html)
