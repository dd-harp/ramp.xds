# Compute the stable orbit for a discrete time system

Run a discrete time system for `Tburn` years to get the stable orbit

## Usage

``` r
dts_stable_orbit(xds_obj, Tburn = 10, yr = 365)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- Tburn:

  the number of steps to burn

- yr:

  the number of time steps in a year

## Value

an **`xds`** object
