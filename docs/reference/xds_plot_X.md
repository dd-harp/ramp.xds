# Basic plotting for epidemiological models

Basic plotting for epidemiological models

## Usage

``` r
xds_plot_X(xds_obj, i = 1, clrs = "black", llty = 1, add = FALSE)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

- clrs:

  a vector of colors

- llty:

  an integer (or integers) to set the `lty` for plotting

- add:

  plot axes only if FALSE
