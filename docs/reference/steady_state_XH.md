# Steady States for **XH**

Compute the steady states as a function of the daily FoI for a static
value of human population density

## Usage

``` r
steady_state_XH(foi, xds_obj, i = 1)
```

## Arguments

- foi:

  the daily FoI

- xds_obj:

  an **`xds`** model object

- i:

  the vector species index

## Value

steady states
