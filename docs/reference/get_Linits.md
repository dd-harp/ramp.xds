# Get Initial Values for the **L** Component

Get Initial Values for the **L** Component

## Usage

``` r
get_Linits(pars, s = 1)
```

## Arguments

- pars:

  an **`xds`** object

- s:

  the species index

## Value

a named [list](https://rdrr.io/r/base/list.html)

## Note

This method dispatches on the class of `pars$Lpar`.
