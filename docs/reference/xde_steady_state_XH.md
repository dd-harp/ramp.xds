# Compute Steady States for an **XH**-Component Module

Compute the steady states as a function of the daily FoI for a model
that has a steady state human / host population density

This method dispatches on the type of `Xpar`.

## Usage

``` r
xde_steady_state_XH(foi, Xpar)
```

## Arguments

- foi:

  the daily FoI

- Xpar:

  a list that defines an \\\cal X\\ model

## Value

none
