# Set up Poisson Exposure and Infection for `dts`

Set up the Poisson model for exposure for discrete time systems

## Usage

``` r
# S3 method for dts
setup_exposure_pois(pars)
```

## Arguments

- pars:

  an **`xds`** object

## Value

an **`xds`** object
