# Set static blood feeding search weights

Set the blood feeding search weights, \\\omega\\, for a set of host
strata

## Usage

``` r
change_blood_weights(pars, search_weights = 1, s = 1, i = 1)
```

## Arguments

- pars:

  an `xds` object

- search_weights:

  the blood feeding search weights

- s:

  the vector species index

- i:

  the host species index

## Value

an `xds` object
