# Get habitat matrix, \\N\\

Output the habitat membership information as a list

## Usage

``` r
get_habitat_matrix(xds_obj, s)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

a [matrix](https://rdrr.io/r/base/matrix.html)
