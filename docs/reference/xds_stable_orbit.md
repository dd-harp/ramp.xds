# Compute stable orbits

Run the system for 10 years, and save the last year's orbits.

## Usage

``` r
xds_stable_orbit(xds_obj, Y = 10, tol = 1e-05)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- Y:

  the number of years to run once

- tol:

  the desired accuracy

## Value

an **`xds`** object
