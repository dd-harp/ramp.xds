# Setup Mosquito Dispersal Matrix

Setup Mosquito Dispersal Matrix

## Usage

``` r
setup_K_matrix(Kname, s, xds_obj, options = list())
```

## Arguments

- Kname:

  a matrix or a string

- s:

  the vector species index

- xds_obj:

  an **`xds`** model object

- options:

  a list of options to configure K_matrix

## Value

an **`xds`** model object
