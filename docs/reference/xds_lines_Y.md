# Add lines for the density of infected and infective mosquitoes

Add lines for the density of infected and infective mosquitoes

## Usage

``` r
xds_lines_Y(xds_obj, s = 1, clrs = "purple", llty = 1)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

- clrs:

  a vector of colors

- llty:

  an integer (or integers) to set the `lty` for plotting
