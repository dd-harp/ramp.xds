# Compute the prevalence of infection by PCR

Implements [F_pr](https://dd-harp.github.io/ramp.xds/reference/F_pr.md)
for the trivial model.

## Usage

``` r
# S3 method for trivial
F_pr_by_pcr(vars, Xpar)
```

## Arguments

- vars:

  a list with the variables attached by name

- Xpar:

  a list defining a model for human

## Value

a [numeric](https://rdrr.io/r/base/numeric.html) vector numeric(0)
