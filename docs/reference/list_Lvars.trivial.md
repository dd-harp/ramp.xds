# List **L** Component Variables for `trivial`

This method dispatches on the type of `pars$Lpar[[s]]`

## Usage

``` r
# S3 method for trivial
list_Lvars(y, pars, s)
```

## Arguments

- y:

  the variables

- pars:

  an **`xds`** object

- s:

  the vector species index

## Value

an empty [list](https://rdrr.io/r/base/list.html)
