# Setup MY_obj for the macdonald model

Implements
[setup_MY_obj](https://dd-harp.github.io/ramp.xds/reference/setup_MY_obj.md)
for the macdonald model

## Usage

``` r
# S3 method for class 'macdonald'
setup_MY_obj(MYname, xds_obj, s, options = list())
```

## Arguments

- MYname:

  the name of the model

- xds_obj:

  an **`xds`** model object

- s:

  the species index

- options:

  a [list](https://rdrr.io/r/base/list.html)

## Value

a [list](https://rdrr.io/r/base/list.html) vector
