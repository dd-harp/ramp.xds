# Compute the get_HTC for the SIS model

Implements `get_HTC` for the SIS model with demography.

## Usage

``` r
# S3 method for class 'SIS'
get_HTC(xds_obj, i)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

## Value

a [numeric](https://rdrr.io/r/base/numeric.html) vector
