# Compute the HTC for the hMoI model

Implements [HTC](https://dd-harp.github.io/ramp.xds/reference/HTC.md)
for the hMoI model with demography.

## Usage

``` r
# S3 method for class 'hMoI'
HTC(xds_obj, i)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

## Value

a [numeric](https://rdrr.io/r/base/numeric.html) vector
