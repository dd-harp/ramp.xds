# Return the variables as a list

This method dispatches on the type of `xds_obj$MY_obj[[s]]`

## Usage

``` r
# S3 method for class 'SI'
get_MY_vars(y, xds_obj, s)
```

## Arguments

- y:

  the variables

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

a [list](https://rdrr.io/r/base/list.html)
