# Set mean forcing

Set the mean daily EIR for an `eir` model

## Usage

``` r
# S3 method for class 'eir'
change_mean_forcing(X, xds_obj, s = 1)
```

## Arguments

- X:

  the new mean forcing parameter

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

a **`ramp.xds`** model object
