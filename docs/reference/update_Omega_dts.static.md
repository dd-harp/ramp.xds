# Update Omega for dts

The static case for
[update_Omega_dts](https://dd-harp.github.io/ramp.xds/reference/update_Omega_dts.md)

## Usage

``` r
# S3 method for class 'static'
update_Omega_dts(xds_obj, s)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  vector species index

## Value

an **xds** model object
