# XH modules

A List of Modules for the **XH**-Component

## In **`ramp.xds`**

- `trivial` - the trivial module: see
  [trivial_XH](https://dd-harp.github.io/ramp.xds/reference/trivial_XH.md)

- `SIS` - an **SIS** compartmental model: see
  [SIS](https://dd-harp.github.io/ramp.xds/reference/SIS.md)

- `hMoI` - a hybrid model for exposure: see
  [hMoI](https://dd-harp.github.io/ramp.xds/reference/hMoI.md)

## In **`ramp.library`**

- **Malaria**

  - `garki` - an implementation of the Garki Model for malaria

  - `SIP` - a compartmental model with treatment and chemoprotection

  - `SIPd` - an implementation of the Garki Model for malaria

- **Compartmental Models**

  - `SIR` - an **SIR** compartmental model

  - `SIRS` - an **SIRS** compartmental model

  - `SEIR` - an **SEIR** compartmental model

  - `SEIRV` - an **SEIRV** compartmental model

  - `SEIS` - an **SEIS** compartmental model

  - `SEISd` - an **SEIS** compartmental model, as a delay differential
    equation
