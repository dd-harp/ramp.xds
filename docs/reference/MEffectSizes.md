# Apply effect sizes (**MY**)

Apply vector control effect sizes to adult mosquito bionomic parameters

## Usage

``` r
MEffectSizes(t, y, xds_obj, s)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

an **`xds`** object
