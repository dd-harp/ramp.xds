# Plot the fraction infective

Plot the fraction infective

## Usage

``` r
xds_plot_Z_fracs(xds_obj, s = 1, clrs = "darkred", llty = 1, add = FALSE)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

- clrs:

  a vector of colors for infective mosquitoes

- llty:

  an integer (or integers) to set the `lty` for plotting

- add:

  do not plot axes if TRUE
