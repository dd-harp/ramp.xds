# parameters for make_function

Return an object to configure a function
[make_function.sin](https://dd-harp.github.io/ramp.xds/reference/make_function.sin.md)

## Usage

``` r
makepar_F_sin(phase = 0, bottom = 0, pw = 1, norm = 365, N = 1)
```

## Arguments

- phase:

  the phase for a seasonal function

- bottom:

  shape parameter

- pw:

  shape parameter

- norm:

  the normalization period

- N:

  the length of the vector to return

## Value

a function for seasonality
