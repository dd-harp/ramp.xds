# Setup a Human Fraction Bionomic Object

Set up an object to compute
[F_maturation](https://dd-harp.github.io/ramp.xds/reference/F_maturation.md),
the maximum maturation rate \\psi\\

## Usage

``` r
setup_psi_obj(psi, L_obj)
```

## Arguments

- psi:

  the human fraction

- L_obj:

  an **`MY`** model object

## Value

a **`MY`** model object
