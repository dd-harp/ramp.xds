# Update Time Away (setup)

Acknowledges a one-time update to `time_away` and sets the port back to
`"static"`.

## Usage

``` r
# S3 method for class 'setup'
update_time_away(xds_obj, i)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

## Value

an **`xds`** object
