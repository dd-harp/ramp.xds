# Change Visitor's NI

Set the value of the parameter describing available visitors

## Usage

``` r
change_visitor_kappa(visitor_kappa, xds_obj, s)
```

## Arguments

- visitor_kappa:

  net infectiousness of visitors

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

an **`xds`** object
