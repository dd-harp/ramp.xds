# Baseline Bionomics for an **L** Component Module

Handle immature mosquito bionomic parameters as a baseline modified by
control

## Usage

``` r
LBaseline(t, y, xds_obj, s)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

- s:

  the species index

## Value

an `xds` object
