# Set **L** Component parameters for `basicL`

Set the values of **L** component parameters

- `psi` or \\\psi\\

- `xi` or \\\xi\\

- `phi` or \\\phi\\

- `theta` or \\\theta\\

## Usage

``` r
# S3 method for class 'basicL'
change_L_pars(xds_obj, s = 1, options = list())
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

- options:

  a named list

## Value

an **`xds`** object
