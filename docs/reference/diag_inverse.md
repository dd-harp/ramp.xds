# Invert a diagonal matrix

Invert a diagonal matrix which is passed as a vector. If any elements
are zero, set them to one.

## Usage

``` r
diag_inverse(x)
```

## Arguments

- x:

  a [numeric](https://rdrr.io/r/base/numeric.html) vector

## Value

a diagonal [matrix](https://rdrr.io/r/base/matrix.html)
