# Setup the Travel EIR

Setup a model for the dEIR experienced while traveling

## Usage

``` r
setup_F_travel_eir(setup_name, xds_obj, i, options)
```

## Arguments

- setup_name:

  a string to dispatch `setup_F_travel_eir`

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

- options:

  a named list to set up `F_travel_eir`

## Value

an **`xds`** model object
