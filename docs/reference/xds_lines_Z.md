# Add lines for the density of infected and infective mosquitoes

Add lines for the density of infected and infective mosquitoes

## Usage

``` r
xds_lines_Z(MY, xds_obj, clrs = "darkred", llty = 1)
```

## Arguments

- MY:

  xds_objed ouptuts

- xds_obj:

  an **`xds`** model object

- clrs:

  a vector of colors

- llty:

  an integer (or integers) to set the `lty` for plotting
