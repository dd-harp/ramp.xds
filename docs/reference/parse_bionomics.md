# Pull bionomic parameters

Pull the stored values of the baseline and modified values of
parameters. Note that the total effect size of control is the ratio.

## Usage

``` r
parse_bionomics(xds_obj, s = 1)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  vector species index

## Value

an **`xds`** model object
