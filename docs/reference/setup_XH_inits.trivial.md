# Setup Xinits.trivial

Implements
[setup_XH_inits](https://dd-harp.github.io/ramp.xds/reference/setup_XH_inits.md)
for the trivial model

## Usage

``` r
# S3 method for class 'trivial'
setup_XH_inits(xds_obj, H, i = 1, options = list())
```

## Arguments

- xds_obj:

  an **`xds`** model object

- H:

  initial host population density

- i:

  the host species index

- options:

  a [list](https://rdrr.io/r/base/list.html)

## Value

a [list](https://rdrr.io/r/base/list.html) vector
