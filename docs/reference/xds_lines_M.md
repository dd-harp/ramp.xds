# Add lines for adult mosquito population density

Add lines for adult mosquito population density

## Usage

``` r
xds_lines_M(xds_obj, s = 1, clrs = "darkblue", llty = 1)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

- clrs:

  a vector of colors

- llty:

  an integer (or integers) to set the `lty` for plotting
