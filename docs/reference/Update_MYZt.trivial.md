# Handle state updates for the `trivial` **MYZ** module

Implements
[Update_MYZt](https://dd-harp.github.io/ramp.xds/reference/Update_MYZt.md)
for the trivial (forced emergence) model.

## Usage

``` r
# S3 method for trivial
Update_MYZt(t, y, pars, s)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- pars:

  a [list](https://rdrr.io/r/base/list.html)

- s:

  the species index

## Value

a null vector
