# Update inits for macdonald adult mosquito model

Update inits for macdonald adult mosquito model

## Usage

``` r
# S3 method for macdonald
update_MYZinits(pars, y0, s)
```

## Arguments

- pars:

  a [list](https://rdrr.io/r/base/list.html)

- y0:

  a vector of variable values from a simulation

- s:

  the species index

## Value

a [list](https://rdrr.io/r/base/list.html)
