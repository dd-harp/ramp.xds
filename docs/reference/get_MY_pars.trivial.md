# Return the parameters as a list

Return the parameters in the trivial **MY**-Component model as a named
list

## Usage

``` r
# S3 method for class 'trivial'
get_MY_pars(xds_obj, s = 1)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

the parameters as a named list
