# Compute other variables at time t

Compute everything but the derivatives for the generalized spatial
differential equation model

## Usage

``` r
reset_state_i(i, tm, y_matrix, pars)
```

## Arguments

- i:

  current simulation time

- tm:

  current simulation time

- y_matrix:

  state vector

- pars:

  a [list](https://rdrr.io/r/base/list.html)

## Value

**pars** a [list](https://rdrr.io/r/base/list.html)
