# Set the initial values to the last values of the last simulation

Set the initial values to the last values of the last simulation

## Usage

``` r
xds_flatten(vars)
```

## Arguments

- vars:

  a [list](https://rdrr.io/r/base/list.html)

## Value

`vars` as an unnamed [vector](https://rdrr.io/r/base/vector.html)
