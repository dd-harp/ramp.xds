# Update State Variables for `basicL` (**L** Component)

Implements
[Update_Lt](https://dd-harp.github.io/ramp.xds/reference/Update_Lt.md)
for the `basicL` competition model.

## Usage

``` r
# S3 method for class 'basicL'
Update_Lt(t, y, xds_obj, s)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

- s:

  the species index

## Value

a [numeric](https://rdrr.io/r/base/numeric.html) vector
