# Set up the negative binomial model of exposure for continuous time models

Set up the negative binomial model of exposure for continuous time
models

## Usage

``` r
# S3 method for xde
setup_exposure_nb(pars, sz)
```

## Arguments

- pars:

  an `xds` object

- sz:

  the negative binomial size parameter (see
  [pnbinom](https://rdrr.io/r/stats/NegBinomial.html))

## Value

an **`xds`** object
