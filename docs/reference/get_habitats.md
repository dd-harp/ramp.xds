# Get the habitat membership vector

Output the habitat membership information as a list

## Usage

``` r
get_habitats(xds_obj, s)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

a [list](https://rdrr.io/r/base/list.html)
