# View habitat membership, \\N\\

Output the habitat membership information as a list

## Usage

``` r
view_habitat_matrix(xds_obj)
```

## Arguments

- xds_obj:

  an **`xds`** model object

## Value

a [list](https://rdrr.io/r/base/list.html)
