# Put Xvars in place of the X variables in y

This method dispatches on the type of `pars$Xpar[[s]]`.

## Usage

``` r
put_Xvars(Xvars, y, pars, i)
```

## Arguments

- Xvars:

  the X variables to insert into y

- y:

  the variables

- pars:

  an **`xds`** object

- i:

  the host species index

## Value

a [list](https://rdrr.io/r/base/list.html)
