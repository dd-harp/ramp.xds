# Plot adult mosquito population density

Plot adult mosquito population density

## Usage

``` r
xds_plot_M(xds_obj, s = 1, clrs = "darkblue", llty = 1, add = FALSE)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

- clrs:

  a vector of colors

- llty:

  an integer (or integers) to set the `lty` for plotting

- add:

  do not plot axes if TRUE
