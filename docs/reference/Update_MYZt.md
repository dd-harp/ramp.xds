# Update States for a Adult Mosquito Model

This method dispatches on the type of `pars$MYZpar[[s]]`.

## Usage

``` r
Update_MYZt(t, y, pars, s)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- pars:

  a [list](https://rdrr.io/r/base/list.html)

- s:

  the species index

## Value

Updated states for an adult mosquito model, a
[vector](https://rdrr.io/r/base/vector.html)
