# Get indices (**L**)

The indices are stored on the **L** object as a named list called `ix`;
`get_L_ix` returns that list.

## Usage

``` r
get_L_ix(xds_obj, s = 1)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

a named [list](https://rdrr.io/r/base/list.html) of indices for the
**L** variables
