# Get spline interpolation points

Get spline interpolation points when `forced_by` = "Lambda"

## Usage

``` r
# S3 method for class 'Lambda'
get_spline_s(xds_obj, s = 1)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

a list, the value of `trend_par`
