# Parse outputs after solving

Parse outputs after solving

## Usage

``` r
parse_orbits(outputs, xds_obj)
```

## Arguments

- outputs:

  a [matrix](https://rdrr.io/r/base/matrix.html) of \_orbits returned by
  deSolve

- xds_obj:

  an **`xds`** model object

## Value

a named [list](https://rdrr.io/r/base/list.html)
