# parse the outputs of an object created by xde_solve or dts_solve

parse the outputs of an object created by xde_solve or dts_solve

## Usage

``` r
parse_orbits(outputs, xds_obj)
```

## Arguments

- outputs:

  a [matrix](https://rdrr.io/r/base/matrix.html) of \_orbits returned by
  deSolve

- xds_obj:

  an **`xds`** model object

## Value

varslist a [list](https://rdrr.io/r/base/list.html)
