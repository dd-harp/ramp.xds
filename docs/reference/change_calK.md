# Change the time spent matrix

Change the time spent matrix

## Usage

``` r
change_calK(calK, pars, s = 1)
```

## Arguments

- calK:

  a mosquito dispersal [matrix](https://rdrr.io/r/base/matrix.html)

- pars:

  a [list](https://rdrr.io/r/base/list.html)

- s:

  the vector species index

## Value

a [list](https://rdrr.io/r/base/list.html)
