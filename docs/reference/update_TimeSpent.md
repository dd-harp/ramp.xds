# Update the time spent Matrix

Port function for the time spent matrix, \\\Theta\\. Dispatches on
`class(xds_obj$XY_interface$timespent_obj[[i]])`.

## Usage

``` r
update_timespent(xds_obj, i)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

## Value

an **`xds`** object
