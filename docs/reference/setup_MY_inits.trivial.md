# Setup Inits for `trivial` (**MY**)

Return the **`xds`** object unmodified

## Usage

``` r
# S3 method for class 'trivial'
setup_MY_inits(xds_obj, s, options = list())
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the species index

- options:

  a [list](https://rdrr.io/r/base/list.html)

## Value

an **`xds`** object
