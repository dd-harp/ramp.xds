# Get Omega

Return the demographic matrix, called \\\Omega\\.

## Usage

``` r
get_Omega(xds_obj, s = 1)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

the demographic matrix
