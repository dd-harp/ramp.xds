# Generalized spatial differential equation model

Compute exogenous forcing variables

## Usage

``` r
# S3 method for full
Exogenous(t, y, pars)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- pars:

  an **`xds`** object

## Value

a [list](https://rdrr.io/r/base/list.html) containing the vector of all
state derivatives
