# Change Mosquito Dispersal Matrix

Change Mosquito Dispersal Matrix

## Usage

``` r
change_K_matrix(K_matrix, xds_obj, s = 1)
```

## Arguments

- K_matrix:

  a mosquito dispersal [matrix](https://rdrr.io/r/base/matrix.html)

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

an **`xds`** model object
