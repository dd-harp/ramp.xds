# Change the egg laying rate

Change the egg laying rate for a static model

## Usage

``` r
change_nu(nu, xds_obj, s)
```

## Arguments

- nu:

  the egg laying rate (# batches, per mosquito, per day)

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

an **`xds`** model object
