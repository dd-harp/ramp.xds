# Compute EIR for each vector-host pair

This function computes the EIR from each vector species for each stratum
of each host species

## Usage

``` r
make_EIR_full(t, y, pars)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- pars:

  an `xds` object

## Value

an `xds` object
