# Get initial values (**L**)

The initial values are stored on the **L** object as a named list called
`inits`; `get_L_inits` returns that list.

## Usage

``` r
get_L_inits(xds_obj, s = 1)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the species index

## Value

a named [list](https://rdrr.io/r/base/list.html)
