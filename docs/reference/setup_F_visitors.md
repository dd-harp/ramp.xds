# Set up no visitors

Setup a model for no time spent visitorsing and no exposure while
visitorsing

## Usage

``` r
setup_F_visitors(mod_name, xds_obj, s, options)
```

## Arguments

- mod_name:

  the model name

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

- options:

  a named list to setup `F_visitors`

## Value

an **`xds`** model object
