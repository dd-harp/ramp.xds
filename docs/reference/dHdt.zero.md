# Compute Demographic Changes

Return a vector of zeros

## Usage

``` r
# S3 method for zero
dHdt(t, y, Hpar)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- Hpar:

  a [list](https://rdrr.io/r/base/list.html)

## Value

a [numeric](https://rdrr.io/r/base/numeric.html) vector of 0s
