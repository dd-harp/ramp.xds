# Setup the Habitats Object

During basic setup, the habitats membership matrix is set up.

The `habitat_obj` handles:

- dynamically changing habitat weights

- availability of bad habitats

- with multiple vector species, weights can be assigned to bad habitats

## Usage

``` r
setup_habitat_object(xds_obj)
```

## Arguments

- xds_obj:

  an **`xds`** model object

## Value

an **`xds`** model object
