# Compute the VC Matrix

Compute vectorial capacity for the `GeM` module

## Usage

``` r
# S3 method for class 'GeM'
dVdt(t, y, xds_obj, i)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

## Value

a numeric [matrix](https://rdrr.io/r/base/matrix.html)
