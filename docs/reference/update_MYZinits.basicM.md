# Make inits for RM adult mosquito model

Make inits for RM adult mosquito model

## Usage

``` r
# S3 method for basicM
update_MYZinits(pars, y0, s)
```

## Arguments

- pars:

  a [list](https://rdrr.io/r/base/list.html)

- y0:

  a vector of variable values from a simulation

- s:

  the species index

## Value

none
