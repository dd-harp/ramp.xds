# Generic Methods for the L Component

A list of generic

- [`dLdt`](https://dd-harp.github.io/ramp.xds/reference/dLdt.md) –
  compute derivatives

- [`Update_Lt`](https://dd-harp.github.io/ramp.xds/reference/Update_Lt.md)
  – update state variables

- [`LBaseline`](https://dd-harp.github.io/ramp.xds/reference/LBaseline.md)
  – set baseline bionomic parameters

- [`LBionomics`](https://dd-harp.github.io/ramp.xds/reference/LBionomics.md)
  – compute bionomic parameters

- [`F_emerge`](https://dd-harp.github.io/ramp.xds/reference/F_emerge.md)
  – compute emergent adults

- [`setup_L_obj`](https://dd-harp.github.io/ramp.xds/reference/setup_L_obj.md)
  – set up the L module object

- [`setup_L_ix`](https://dd-harp.github.io/ramp.xds/reference/setup_L_ix.md)
  – set index values

- [`setup_L_inits`](https://dd-harp.github.io/ramp.xds/reference/setup_L_inits.md)
  – set up initial values

- [`get_L_inits`](https://dd-harp.github.io/ramp.xds/reference/get_L_inits.md)
  – get initial values

- [`change_L_inits`](https://dd-harp.github.io/ramp.xds/reference/change_L_inits.md)
  – change initial values

- [`get_L_vars`](https://dd-harp.github.io/ramp.xds/reference/get_L_vars.md)
  – list state variables

- [`get_L_pars`](https://dd-harp.github.io/ramp.xds/reference/get_L_pars.md)
  – get parameters

- [`change_L_pars`](https://dd-harp.github.io/ramp.xds/reference/change_L_pars.md)
  – change parameters

- [`parse_L_orbits`](https://dd-harp.github.io/ramp.xds/reference/parse_L_orbits.md)
  – parse outputs

- [`get_L_orbits`](https://dd-harp.github.io/ramp.xds/reference/get_L_orbits.md)
  – get saved orbits

- [`steady_state_L`](https://dd-harp.github.io/ramp.xds/reference/steady_state_L.md)
  – compute steady states

- [`check_L`](https://dd-harp.github.io/ramp.xds/reference/check_L.md) –
  run consistency checks

- [`skill_set_L`](https://dd-harp.github.io/ramp.xds/reference/skill_set_L.md)
  – return module skill set
