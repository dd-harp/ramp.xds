# Set the values of exogenous variables

After none setup, no exogenous variables are configured so traps returns
the **`xds`** object without modification

## Usage

``` r
# S3 method for class 'none'
Traps(t, y, xds_obj)
```

## Arguments

- t:

  current simulation time

- y:

  state variables

- xds_obj:

  an **`xds`** model object

## Value

an **`xds`** object
