# Change shock parameters

Change parameters for the shock function

## Usage

``` r
change_shock(shock_par, xds_obj, s = 1)
```

## Arguments

- shock_par:

  parameters for
  [make_function](https://dd-harp.github.io/ramp.xds/reference/make_function.md)

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

an **`xds`** object
