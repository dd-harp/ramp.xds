# Adult Mosquito Bionomics - Modified by Control

Modify the baseline adult mosquito bionomic parameters

## Usage

``` r
MBionomics(t, y, xds_obj, s)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

- s:

  the species index

## Value

an `xds` object
