# Setup Initial Values for **V** Modules

This method dispatches on `xds_obj$V_obj[[i]]`.

## Usage

``` r
setup_V_inits(xds_obj, H, i, options = list())
```

## Arguments

- xds_obj:

  an **`xds`** model object

- H:

  initial host population density

- i:

  the host species index

- options:

  a [list](https://rdrr.io/r/base/list.html)

## Value

an **`xds`** object
