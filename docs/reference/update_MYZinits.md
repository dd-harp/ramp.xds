# Set the initial values as a vector

This method dispatches on the type of `pars$MYZpar`.

## Usage

``` r
update_MYZinits(pars, y0, s)
```

## Arguments

- pars:

  a [list](https://rdrr.io/r/base/list.html)

- y0:

  a vector of variable values from a simulation

- s:

  the species index

## Value

a [list](https://rdrr.io/r/base/list.html)
