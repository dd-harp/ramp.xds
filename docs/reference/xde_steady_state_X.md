# Compute Steady States for an **X**-Component Module

Compute the steady states as a function of the daily FoI for a static
value of human population density

## Usage

``` r
xde_steady_state_X(foi, H, Xpar)
```

## Arguments

- foi:

  the daily FoI

- H:

  human / host population density

- Xpar:

  a list that defines an **X** model object

## Value

none
