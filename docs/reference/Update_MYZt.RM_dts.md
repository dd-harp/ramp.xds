# Derivatives for adult mosquitoes

Implements
[Update_MYZt](https://dd-harp.github.io/ramp.xds/reference/Update_MYZt.md)
for the RM_dts model.

## Usage

``` r
# S3 method for RM_dts
Update_MYZt(t, y, pars, s)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- pars:

  a [list](https://rdrr.io/r/base/list.html)

- s:

  the species index

## Value

a [numeric](https://rdrr.io/r/base/numeric.html) vector
