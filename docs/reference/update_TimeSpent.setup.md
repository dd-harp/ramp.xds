# Update the time spent Matrix (setup)

Acknowledges a one-time update to the time spent matrix and sets the
port back to `"static"`.

## Usage

``` r
# S3 method for class 'setup'
update_timespent(xds_obj, i)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

## Value

an **`xds`** object
