# Get orbits (**MY**)

After solving, the orbits are parsed and stored. Get orbits retrieves
the parsed outputs for the **MY** module

## Usage

``` r
get_MY_orbits(xds_obj, s = 1)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

the orbits for the **MY** component
