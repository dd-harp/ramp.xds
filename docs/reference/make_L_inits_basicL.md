# Make initial values for `basicL` (**L** component)

Initial values of the variable \\L\\ can be set by passing `L` in
`options`

## Usage

``` r
make_L_inits_basicL(nHabitats, options = list(), L = 1)
```

## Arguments

- nHabitats:

  the number of habitats in the model

- options:

  a [list](https://rdrr.io/r/base/list.html) that overwrites default
  values

- L:

  initial conditions

## Value

a [list](https://rdrr.io/r/base/list.html) with Linits added
