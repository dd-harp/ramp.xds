# Static model patch emigration

Implements
[F_sigma](https://dd-harp.github.io/ramp.xds/reference/F_sigma.md) for a
static model

## Usage

``` r
# S3 method for class 'static'
F_sigma(t, xds_obj, s)
```

## Arguments

- t:

  current simulation time

- xds_obj:

  an **`xds`** model object

- s:

  vector species index

## Value

\\sigma\\, the patch emigration rate
