# Set up (or change) a Time Spent matrix

Set up (or change) a Time Spent matrix

## Usage

``` r
change_TimeSpent_matrix(TimeSpent, xds_obj, i = 1)
```

## Arguments

- TimeSpent:

  a time spent matrix

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

## Value

a modified **`xds`** model object
