# Adult Mosquito Bionomics - Baseline

Compute the baseline adult mosquito bionomic parameter values

## Usage

``` r
MBaseline(t, y, xds_obj, s)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

- s:

  the species index

## Value

an `xds` object
