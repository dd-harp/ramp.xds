# Set yy

Set the yy for the seasonal pattern for an `eir` model and return the
**`ramp.xds`** model object

## Usage

``` r
# S3 method for class 'eir'
change_spline_y(yy, xds_obj, s = 1)
```

## Arguments

- yy:

  new y values for the interpolation points

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

a **`ramp.xds`** model object
