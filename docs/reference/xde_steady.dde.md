# Solve for the steady state of a system of equations using [rootSolve::steady](https://rdrr.io/pkg/rootSolve/man/steady.html)

This method dispatches on the type of `pars$xde`

## Usage

``` r
# S3 method for dde
xde_steady(pars)
```

## Arguments

- pars:

  an **`xds`** object

## Value

an **`xds`** object
