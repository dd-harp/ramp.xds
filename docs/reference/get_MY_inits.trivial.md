# Get inits for `trivial` (MY)

Return a [numeric](https://rdrr.io/r/base/numeric.html) vector of length
0

## Usage

``` r
get_MY_inits.trivial(xds_obj, s)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

a [numeric](https://rdrr.io/r/base/numeric.html) vector of length 0
