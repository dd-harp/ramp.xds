# Set up no habitats

Setup a model for no time spent searching and no exposure while
searching

## Usage

``` r
setup_F_habitat_weights(mod_name, xds_obj, s, options)
```

## Arguments

- mod_name:

  the model name

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

- options:

  a named list to configure `F_habitat_weights`

## Value

an **`xds`** model object
