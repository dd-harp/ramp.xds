# Set up no blood_hosts

Setup a model for no time spent blood_hostsing and no exposure while
blood_hostsing

## Usage

``` r
# S3 method for class 'ts_func'
setup_blood_hosts(mod_name, xds_obj, s, options)
```

## Arguments

- mod_name:

  the model name

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

- options:

  a list to configure `F_blood_hosts`

## Value

an **`xds`** model object
