# Make a time spent matrix, called TimeSpent

Make a time spent matrix, called TimeSpent

## Usage

``` r
setup_TimeSpent(TimeSpent, xds_obj, i, options = list())
```

## Arguments

- TimeSpent:

  a matrix or a setup function name

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

- options:

  configuration options

## Value

an **`xds`** model object
