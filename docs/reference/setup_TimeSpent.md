# Make a time spent matrix, called timespent

Make a time spent matrix, called timespent

## Usage

``` r
setup_timespent(name, xds_obj, i, options = list())
```

## Arguments

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

- options:

  configuration options

- timespent:

  a matrix or a setup function name

## Value

an **`xds`** object
