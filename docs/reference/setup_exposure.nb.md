# Set up a nbson model for Exposure and Infection

Set up a nbson model for Exposure and Infection

## Usage

``` r
# S3 method for class 'nb'
setup_exposure(EHname, xds_obj, i = 1, options = list())
```

## Arguments

- EHname:

  environmental heterogeneity model name

- xds_obj:

  an **`xds`** model object

- i:

  the host species index

- options:

  set up options list

## Value

an **`xds`** object
