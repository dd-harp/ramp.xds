# Return initial values as a list

Implements
[get_MYZinits](https://dd-harp.github.io/ramp.xds/reference/get_MYZinits.md)
for the GeRM model.

## Usage

``` r
# S3 method for GeRM
get_MYZinits(pars, s = 1)
```

## Arguments

- pars:

  a [list](https://rdrr.io/r/base/list.html)

- s:

  the species index

## Value

[list](https://rdrr.io/r/base/list.html)
