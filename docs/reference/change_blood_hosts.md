# Set static blood feeding search weights

Set the blood feeding search weights, \\\omega\\, for a set of host
strata

## Usage

``` r
change_blood_hosts(blood_hosts, xds_obj, s)
```

## Arguments

- blood_hosts:

  availability of other blood hosts

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

an `xds` object
