# Static model patch emigration

Implements
[F_eip](https://dd-harp.github.io/ramp.xds/reference/F_eip.md) for a
static model

## Usage

``` r
# S3 method for class 'static'
F_eip(t, xds_obj, s)
```

## Arguments

- t:

  current simulation time

- xds_obj:

  an **`xds`** model object

- s:

  vector species index

## Value

\\eip\\, the patch emigration rate
