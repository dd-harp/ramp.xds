# Get Blood Feeding Search Weights

Get Blood Feeding Search Weights

## Usage

``` r
get_search_weights_blood(xds_obj, s = 1, i = 1)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

- i:

  the host species index

## Value

a [numeric](https://rdrr.io/r/base/numeric.html) vector of length
`nStrata`
