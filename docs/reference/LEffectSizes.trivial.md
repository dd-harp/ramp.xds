# Apply effect sizes for `trivial` (**L**)

Return the **`xds`** object unmodified

## Usage

``` r
# S3 method for class 'trivial'
LEffectSizes(t, y, xds_obj, s)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

- s:

  the species index

## Value

an **`xds`** object
