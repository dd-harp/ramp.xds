# Set up the negative binomial model for exposure for discrete time models

Set up the negative binomial model for exposure for discrete time models

## Usage

``` r
# S3 method for dts
setup_exposure_nb(pars, sz)
```

## Arguments

- pars:

  an `xds` object

- sz:

  the negative binomial size parameter (see
  [pnbinom](https://rdrr.io/r/stats/NegBinomial.html))

## Value

an **`xds`** object
