# time spent

Compute and store host available, \\W\\, total blood host available,
\\B\\, the time spent matrix \\\Theta\\, and the time-at-risk matrix
\\\Psi\\,

## Usage

``` r
TimeSpent(t, y, xds_obj)
```

## Arguments

- t:

  the time

- y:

  the state variables

- xds_obj:

  an **`xds`** model object

## Value

an `xds` object
