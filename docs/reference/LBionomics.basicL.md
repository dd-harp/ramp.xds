# Bionomics for `basicL` (**L** Component)

Implements
[LBionomics](https://dd-harp.github.io/ramp.xds/reference/LBionomics.md)
for the `basicL`

## Usage

``` r
# S3 method for class 'basicL'
LBionomics(t, y, xds_obj, s)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

- s:

  the species index

## Value

a **`ramp.xds`** object
