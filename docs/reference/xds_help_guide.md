# **`ramp.xds`**: Getting Help

**`ramp.xds`** concepts and functions are well documented. Use these
guides to find a concept or function.

- [xds_help_info](https://dd-harp.github.io/ramp.xds/reference/xds_help_info.md)
  – a guide to help functions for **`ramp.xds`** concepts

- [xds_info_basic_setup](https://dd-harp.github.io/ramp.xds/reference/xds_info_basic_setup.md)
  – an overview of basic setup

- [xds_info_setup_options](https://dd-harp.github.io/ramp.xds/reference/xds_info_setup_options.md)
  – setup for other features

- **`get_`** functions inspect a model component or outputs

- **`change_`** functions modify part of a model component

- **`setup_`** functions replace a model component
