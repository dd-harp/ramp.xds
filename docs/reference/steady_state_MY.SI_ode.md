# Steady States: MY-SI

This method dispatches on the type of `MY_obj`.

## Usage

``` r
# S3 method for class 'SI_ode'
steady_state_MY(Lambda, kappa, xds_obj, s = 1)
```

## Arguments

- Lambda:

  the daily emergence rate of adult mosquitoes

- kappa:

  net infectiousness

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

none
