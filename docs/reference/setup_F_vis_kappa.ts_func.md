# Set

Setup a model for the net infectiousness of visitors

## Usage

``` r
# S3 method for class 'ts_func'
setup_F_vis_kappa(mod_name, xds_obj, s, options)
```

## Arguments

- mod_name:

  the model name

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

- options:

  a list to configure `F_vis_kappa`

## Value

an **`xds`** model object
