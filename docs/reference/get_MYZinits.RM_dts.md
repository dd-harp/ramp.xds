# Return initial values as a vector

Implements
[get_MYZinits](https://dd-harp.github.io/ramp.xds/reference/get_MYZinits.md)
for the RM_dts model.

## Usage

``` r
# S3 method for RM_dts
get_MYZinits(pars, s)
```

## Arguments

- pars:

  a [list](https://rdrr.io/r/base/list.html)

- s:

  the species index

## Value

[list](https://rdrr.io/r/base/list.html)
