# Setup initial values (**MY**)

This method dispatches on `MYname`.

## Usage

``` r
setup_MY_inits(xds_obj, s, options = list())
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

- options:

  a [list](https://rdrr.io/r/base/list.html)

## Value

[list](https://rdrr.io/r/base/list.html)
