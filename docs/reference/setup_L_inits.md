# Setup initial values (**L**)

Sets the initial values of the **L** Component variables from `options`,
a named list.

## Usage

``` r
setup_L_inits(xds_obj, s, options = list())
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the species index

- options:

  a named [list](https://rdrr.io/r/base/list.html)

## Value

an **`xds`** object
