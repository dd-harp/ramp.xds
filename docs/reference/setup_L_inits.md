# Setup Initial Values for the **L** Component

This method dispatches on `Lname`.

## Usage

``` r
setup_L_inits(xds_obj, s, options = list())
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the species index

- options:

  a [list](https://rdrr.io/r/base/list.html)

## Value

an **`xds`** object
