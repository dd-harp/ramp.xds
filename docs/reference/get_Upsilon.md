# Get Upsilon

Return \\\Upsilon\\, a matrix describing survival and dispersal through
the EIP.

## Usage

``` r
get_Upsilon(xds_obj, s = 1)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

the demographic matrix
