# Set up Poisson Exposure and Infection for `xde`

Set up the Poisson model for exposure for continuous time models

## Usage

``` r
# S3 method for xde
setup_exposure_pois(pars)
```

## Arguments

- pars:

  an **`xds`** object

## Value

an **`xds`** object
