# Modify resources and resource availability

Implements
[Resources](https://dd-harp.github.io/ramp.xds/reference/Resources.md)
for the static model of resources

## Usage

``` r
# S3 method for static
Resources(t, pars)
```

## Arguments

- t:

  current simulation time

- pars:

  a [list](https://rdrr.io/r/base/list.html)

## Value

none
